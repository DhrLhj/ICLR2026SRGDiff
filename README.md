# ICLR2026SRGDiff
SRGDiff: a step-aware residual-guided diffusion model for EEG spatial super-resolution. It learns dynamic, step-wise residual conditions from low-density EEG to steer denoising and recover high-density signals with improved fidelity and reduced spatial–spectral shift. Evaluated on SEED/SEED-IV/Localize-MI across scales; code &amp; configs included.
